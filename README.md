Tic-Tac-Toe is a classic two-player game built using HTML, CSS, and JavaScript. 
Players take turns marking X or O on a 3Ã—3 grid. 
The first player to align three marks horizontally, vertically, or diagonally wins the game.

## Features
- Two-player mode (X vs O)
- Winner and draw detection
- Restart game option
- Simple and user-friendly interface

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Play
1. Player X starts the game
2. Players take turns clicking on empty cells
3. The first player to get three in a row wins
4. If all cells are filled with no winner, the game is a draw
5. Click the restart button to play again
